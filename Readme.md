Skool Monthly Yearbook Builder
A browser-based page builder for creating monthly yearbook pages for Skool communities.
Features

Top 5 member cards with uploaded AI-generated images
Leaderboard banner image
7-Day spotlight circles with nameplates
Special Thanks and What's Next sections
Custom background color or image
Heading and badge color controls
Download completed page as HTML
Print / PDF export

Usage
Go to the site 
Fill in the sidebar fields and the page preview updates live.
When finished, hit Download Page to save the completed HTML
